\# Hotel Adaptive Pricing (Beginner friendly)



This project builds a Streamlit web app that:

\- Shows customer behavior dashboards (inside the web app)

\- Trains a model automatically (on first run) and saves it

\- Lets users enter booking details and returns a recommended ADR (price)



\## How to run locally

1\. Create a folder and copy files from this repo.

2\. Put your dataset `hotel\_bookings.csv` into the `data/` folder.

3\. Create a virtual environment and install dependencies:

&nbsp;  python -m venv venv

&nbsp;  source venv/bin/activate      # mac/linux
